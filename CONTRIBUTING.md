# Contributing

Thanks for your interest in this training repository.

## Contribution Model

- Use forks and pull requests.
- Direct pushes to `main` are blocked by branch protection.
- Keep changes focused and easy to review.

## For Learners

If you are using this for self-practice:

1. Fork the repository.
2. Configure your own GitHub Actions variables/secrets in your fork.
3. Use your own AWS account/cluster.

## Pull Request Checklist

- [ ] Updated docs for any workflow or behavior changes
- [ ] No secrets or account-specific credentials committed
- [ ] Commands in docs were tested
